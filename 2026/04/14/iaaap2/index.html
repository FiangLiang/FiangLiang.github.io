Chap 2: Binary Image and Morphology

标题名为“二值图像与形态学”，咱先来看看什么是二值图像

1 Binary Images

灰度图与二值图像的差别如下图所示：

二值图像中像素的值只有 {0,1} 或者 {0,255}，编程中我们一般用 {0,255} 来构造二值图像，其优缺点如下：

  • 优点
    • 更小的内存需求
    • 运行速度更快
    • 为二值图像开发的算法往往可以用于灰度级图像
  • 缺点
    • 应用范围有限
    • 无法推广到三维空间中
    • 表现力欠缺，不能表现物体内部细节
    • 无法控制对比度

2 Image Binarization

如何构建一个二值图像，我们可以从一个灰度图像出发

  • 设置一个阈值 Threshold，比阈值小的置为 0, 比阈值大的置为 255

    {l(x,y)=0, if l(x,y)<Thresholdl(x,y)=255, else l(x,y)≥Threshold\begin{cases} l(x,y)=0,\text{ if }l(x,y)<\text{Threshold}\\ l(x,y)=255,\text{ else }l(x,y)\ge\text{Threshold}\\ \end{cases} {l(x,y)=0, if l(x,y)<Thresholdl(x,y)=255, else l(x,y)≥Threshold​

  • 选取合适阈值的基本思想：

    • 将二值图像视为两部分，一部分对应前景（foreground），另一部分对应背景（background）

    • 尝试找到一个合适的阈值，使得前景和背景的内部方差 (variance)（前景与前景，背景与背景）最小化，而让外部方差（前景与背景）最大化

    • 内部协方差和外部协方差可以通过大津算法 (Otsu’s method) 来计算，推导过程略，这里直接给出公式：

      σwithin2(T)=NFNσFgrd2(T)+NBNσBgrd2(T)σbetween2(T)=NF⋅NBN2(μF−μB)2\sigma^2_{within}(T)=\frac{N_F}{N}\sigma^2_{Fgrd}(T)+\frac{N_B}{N}\sigma^2_{Bgrd}(T)\\ \sigma^2_{between}(T)=\frac{N_F\cdot N_B}{N^2}(\mu_F-\mu_B)^2 σwithin2​(T)=NNF​​σFgrd2​(T)+NNB​​σBgrd2​(T)σbetween2​(T)=N2NF​⋅NB​​(μF​−μB​)2

  • 选取合适阈值的具体步骤：

    1. 确定原始图像中像素的最大值和最小值
    2. 令阈值 = 像素最小值 + 1，对原始图像进行二值化操作
    3. 确定前景和背景（小于阈值视为前景，大于阈值视为背景），分别计算当前阈值下的内部协方差和外部协方差，然后阈值 + 1
    4. 重复 2、3 两步，直到阈值达到像素最大值
    5. 找到最大外部协方差和最小内部协方差对应的阈值

  • 阈值策略的缺陷

    • 全局二值化操作不给力

    • 解决方法：局部自适应操作(local adaptive operation)

3 Morphological Operation

这里的形态学（morphology）指的是基于集合论的数学形态学（mathematical morphology），可用于噪声过滤、形状简化、细化、分割、物体描述等

基本操作：膨胀 (dilation)、腐蚀 (erosion)、开操作 (opening) 和闭操作 (closing)

不过在学习这四种基本操作前，我们先简单介绍一下可能用得上的集合论知识

3.2 Dilation

膨胀是将与物体“接触”的所有背景点合并到该物体中，使边界向外部扩张的过程，可以用来填补物体中的空洞

下面是膨胀操作的数学定义，比较抽象

  • 令 AAA 为二值图像，BBB 为二值模板，计算公式为：

    A⊕B={z∣Bz∩A≠∅}A\oplus B=\{z|B_z\cap A\ne \emptyset\} A⊕B={z∣Bz​∩A=∅}

上式表示使得平移后的 B 与 A 的交集不为空的所有 B 的中心位置的集合

3.3 Erosion

腐蚀(erosion) ：一种消除边界点，使边界向内部收缩的过程。可以用来消除小且无意义的物体。

数学定义也比较抽象：

A⊖B={z∣Bz⊆A}A\ominus B=\{z|B_z\sube A\} A⊖B={z∣Bz​⊆A}

膨胀是有交集，而腐蚀需要完全被包含。

3.4 Opening

开操作(opening)：先腐蚀，再膨胀

数学公式为：

A∘B=(A⊖B)⊕BA\circ B=(A\ominus B)\oplus B A∘B=(A⊖B)⊕B

作用：断开较窄的狭颈（细长的白色线条），并消除细小的突出物，使边界更加平滑，并保留了原图的特征

3.5 Closing

闭操作(closing)：先膨胀，再腐蚀

公式为：

A⋅B=(A⊕B)⊖BA \cdot B=(A\oplus B)\ominus B A⋅B=(A⊕B)⊖B

作用：弥合较窄的间断和细长的沟壑，消除小的空洞，填补轮廓线中的断裂，在保留原图特征的同时使边界更平滑
