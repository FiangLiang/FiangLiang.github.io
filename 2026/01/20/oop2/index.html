还是没看懂标题，不过这一章主要在讲 STL

STL，全称为 Standard Template Library， 是 C++ 中一类标准库的统称，包括一部分的输入输出库、常用的数据结构库、常用的简单算法库。下面我们将列举一些 STL 中常用的数据结构和算法。

Container

STL 中的容器（container）实际上就是数据结构的一种实现，可以分为一下三类：

  • Sequence Containers：如 vector、deque、list 等，主要用于存储线性结构
  • Associative Containers：如 set、map 等，主要用于存储键值对数据
  • Container Adapters：如 stack、queue、priority_queue，基于现有序列容器实现，主要用于对其他容器进行封装和适配

Iterators

迭代器（Iterators）是STL中用于遍历容器的对象。以 vector 为例，遍历一个容器通常有如下方法：

  • 下标遍历（仅部分容器可以采用这种方式）

  • 迭代器（绝大多数容器支持这种方式，Container Adapters除外）

  • Range-based For Loop（C++11开始支持，可代替迭代器使用）

Vector

vector 是一个动态数组，可以在运行时自动调整大小。

  • 初始化

  • 元素访问

  • 迭代器

  • 容量管理

  • 修改元素

List

list 是一个双向链表，支持在任意位置插入和删除元素，复杂度为 O(1)O(1)O(1)，但不支持下标访问

  • 初始化

    与 vector 类似

  • 元素访问

    不可通过下标访问，其余类似 vector

  • 迭代器

    与 vector 类似

  • 容量管理

    无法用 reserve 等操作更改大小或者容量

  • 修改元素

Map

map 是一个关联容器，用于存储键值对数据，底层实现通常是红黑树，支持在 O(log⁡n)O(\log n)O(logn) 内完成插入、删除和查找操作。

  • 初始化

  • 元素访问

  • 迭代器

    与 vector 类似

  • 容量管理

    与 list 类似

  • 修改元素
